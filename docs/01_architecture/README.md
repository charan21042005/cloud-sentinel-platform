# 🏛️ System Architecture Documentation

This directory contains the detailed architectural design of the **Cloud Sentinel Platform**.

## 📑 Contents
*   **[System Architecture](System_Architecture.md):** Deep dive into the multi-layer topology, microservices communication, and data flow.

## 🏗️ Design Principles
1.  **High Availability:** Multi-AZ deployment in AWS.
2.  **Scalability:** Independent scaling of frontend and backend pods.
3.  **Security:** Private subnet isolation for database and internal services.
4.  **Observability:** Integrated monitoring from the ground up.
